# WebArcLight Pro (Advanced Replica)

A modern, responsive, conversion-focused replica-style landing page inspired by webarclight.com, with upgraded UI/UX.

## Features
- Premium hero + gradient theme
- Services, process, why-us sections
- Lead capture form
- Basic SEO and Open Graph tags
- Schema.org organization markup
- Fully responsive layout

## Run
Just open `index.html` in a browser.

## Deploy options
- GitHub Pages
- Netlify
- Vercel (static)

## Note
This project is an inspired redesign, not a source-code copy.
